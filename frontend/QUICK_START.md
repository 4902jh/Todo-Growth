# 프론트엔드 빠른 시작 가이드

## 🚀 빠른 실행 방법

### 1. Node.js 설치 확인

```bash
node -v
npm -v
```

Node.js가 없으면: https://nodejs.org/ 에서 다운로드

### 2. 프로젝트 폴더로 이동

```bash
cd frontend
```

### 3. 패키지 설치

```bash
npm install
```

### 4. 개발 서버 실행

```bash
npm start
```

브라우저에서 자동으로 `http://localhost:3000`이 열립니다!

## ⚠️ 주의사항

### 백엔드 서버 실행 필요

프론트엔드를 실행하기 전에 **백엔드 서버가 실행 중**이어야 합니다:

1. `backend-java` 폴더에서 백엔드 실행
2. `http://localhost:8080`에서 서버가 실행 중인지 확인
3. 그 다음 프론트엔드 실행
